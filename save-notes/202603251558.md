# Save Note

## Completed

- Added full user CRUD, trip CRUD, custom exceptions, and controller-facade-service-repository layering.
- Added MapStruct support, Swagger/OpenAPI, and Swagger header authorization support for `accessToken`.
- Removed Spring Session JDBC.
- Added stateless access-token security with:
  - `POST /auth/login`
  - `accessToken` header auth
  - special `master` token support
  - admin seed user via Liquibase
- Added `@UserId` argument injection from the authenticated token and applied it to trip creation.
- Added Liquibase support for PostgreSQL, local/main profile config, timestamp-based changelog naming, and seeded reference data.
- Added `Nationality`, `City`, `Keyword`, and `Trip` domain/schema work.
- Replaced the broken `JsonNullable` integration with the internal `PatchValue` tri-state request wrapper.
- Moved save notes into `save-notes/`.

## Notes

- Protected APIs currently expect the `accessToken` header.
- Swagger UI is public, and secured APIs can use the Swagger authorize lock with `accessToken`.
- User and trip partial updates now use `PatchValue` semantics:
  omitted = no change, `null` = clear, value = update.
